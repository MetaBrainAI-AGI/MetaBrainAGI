# VisionCreatorTools <sub>(Free)</sub>

**The content-creation pack for the VisionPRIME / MetaBrainAGI family.** One pack that
turns a brief into finished media — and a **router** that picks the best tool per job,
checks which key you have, and runs the right sub-skill.

> Free. Open. Part of the MetaBrainAGI family — [metabrainagi.ai](https://metabrainagi.ai).

| Job | What you get |

## How it works

A single **router**: for any content request it ranks the available tools (researched +
ranked June 2026), uses the **#1 tool you have a key for**, falls back down the list, and —
if no key — tells you exactly which key is needed and uses the best free option meanwhile.

Composes the design + marketing skills you already have (`banner-design`, `canvas-design`,
`brand`, `video-editing`, `videodb`, `fal-ai-media`, `market-social`, `content-engine`) with
the AI-media APIs.
